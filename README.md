# Study Focus Scheduler

An Android app built with **Kotlin** to organize study notes and folders, with optional reminders.

## Features

- Create folders and add notes
- Set reminders for notes
- View and manage notes
- Toggle between dark and light modes
- Notes, folders, and reminders saved locally using Room Database
- Built using MVP architecture

## Tech Stack

- Kotlin
- Android Jetpack (Lifecycle, ViewModel, LiveData)
- Room Database
- MVP Architecture
- ViewBinding
- Jetpack Compose (enabled but not fully used)
- Material Design

## Getting Started

1. Clone the repo
2. Open in Android Studio
3. Build and run the app on an emulator or device
